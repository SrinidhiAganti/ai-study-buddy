# AI Study Buddy - Requirements

## Overview
AI Study Buddy is an AI-powered learning assistant for students.

## Functional Requirements
- Users can input study text
- System provides answers
- System generates summaries
- System creates MCQs

## Non-Functional Requirements
- Simple UI
- Fast response
- Web-based access

## Target Users
College students

## Tools
Python Flask, HTML/CSS/JS, Kiro
